## March 2026

### Documentation Site Launch

<Badge type="info" text="March 28, 2026" />

We're excited to announce the launch of our new documentation site! The docs are now fully integrated with the marketing site and feature a beautiful orange theme.

#### ✨ New Features

- **📚 Comprehensive Documentation** - Complete guides, tutorials, and API references
- **🎨 Unified Design** - Seamless integration with the marketing site
- **🌓 Dark & Light Modes** - Switch between themes for comfortable reading
- **🔍 Powerful Search** - Quickly find what you need with local search
- **📱 Mobile Responsive** - Perfect experience on all devices

#### 🎨 Design

- Orange accent color theme matching Hive branding
- Monospace fonts for better code readability
- Clean, modern interface with VitePress

#### 📖 Content

- User Guide with comprehensive instructions
- FAQ with common questions and troubleshooting
- Keyboard Shortcuts reference
- Technical documentation for developers

#### 🔗 Navigation

- Easy access from marketing site
- Click logo to return to homepage
- Organized sidebar with sections
